# 运行隐私节点

如何成为 BOOLNetwork 的隐私节点，参与隐私共识。

---

## 硬件要求

:::tip
隐私节点最重要的是 SGX 环境
:::

建议成为隐私节点的最小硬件环境如下：

- OS: Ubuntu 18.04
- CPU：至少 8 核 （SGX 必须支持）
- RAM：至少 64G 内存，越大越好
- 存储： 至少 2T SSD/HHD，越大越好
- 网络： 独立 IP，至少 100M 带宽，越大越好。
