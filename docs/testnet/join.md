# 参与

参与 BOOLNetwork 测试网，体验不一样的跨链方案。

## 应该什么钱包

BOOLNetwork 链的账户是以太坊兼容的，因此安装了 Metamask 插件的浏览器能与链快速交互。

## 如何获取测试币

官方提供了[水龙头](https://faucet.bool.network/)免费领取测试币。
